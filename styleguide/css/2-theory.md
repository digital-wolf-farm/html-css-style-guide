### Theory

1. Elements of ruleset

    Example ruleset:
    ```css
    p,
    .regular-text {
        font-size: 16px;
        font-weight: 400;
    }
    ```

    Legend:
    * `p` - selector
    * `{ ... }` - declaration block
    * `font-weight: 400;` - declaration
    * `font-size` - property
    * `16px` - property value or just value
